⚡

Continue-on-Success

Always add --continue-on-success (-c) when spraying files. Add --jitter 2-5 for lockout protection. Note: --continue-on-success is incompatible with command execution.

💡

--no-bruteforce Flag

When using -u users.txt -p passwords.txt, add --no-bruteforce to test line 1:1 pair matching. Useful for credential lists from dumps.
