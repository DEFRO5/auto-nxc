⚡

Continue-on-Success

Always add --continue-on-success (-c) when spraying files. Without it nxc stops at the first hit. Add --jitter 2-5 for lockout protection. Note: --continue-on-success is incompatible with command execution — use it for auth checking only.

💡

--no-bruteforce Flag

When using -u users.txt -p passwords.txt, nxc tries every user:pass combo. Add --no-bruteforce to test line 1 of users with line 1 of passwords (1:1 pair matching). Useful for credential lists from dumps.
